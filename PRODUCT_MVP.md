## 1. 产品定位

### 1.1 一句话描述

**面向个人投资者的量化回测工具**

### 1.2 核心价值

| 维度 | 描述 |
|-----|------|
| **做什么** | 帮用户验证投资策略的历史表现 |
| **为谁做** | 想学量化、验证策略的散户 |
| **解决了什么** | 不懂编程也能做回测 |
| **差异化** | 简单、便宜、垂直A股 |

### 1.3 与竞品对比

| 维度 | 聚宽/米筐 | StockQuant Pro |
|-----|----------|----------------|
| 目标用户 | 专业量化 | 普通散户 |
| 学习成本 | 高（需编程） | 低（界面操作） |
| 价格 | 高（500+/月） | 低（99/年） |
| 功能 | 全面 | 聚焦回测 |

### 1.4 双线策略

**线路A：回测工具（对外卖）**
- 量化回测平台
- 用户验证策略
- 自己做决策

**线路B：筛选结果（自己用 + 付费用户）**
- 每日因子筛选
- 改造后合规输出
- 不构成投资建议

## 2. 目标用户

### 2.1 用户画像

**核心用户：**

**次要用户：**

- 个人投资者

### 2.2 用户需求

| 需求 | 优先级 | 满足方式 |
|-----|--------|---------|
| 验证策略效果 | P0 | 回测功能 |
| 学习量化知识 | P1 | 因子分析 |

## 3. 核心功能

### 3.1 功能矩阵

| 功能模块 | 免费 | 专业版（99元/年） |
|---------|------|------------------|
| **策略回测** | 3次/月 | 无限 |
| **回测周期** | 最多1年 | 无限 |
| **因子选择** | 10个基础因子 | 52个全部因子 |
| **回测报告** | 基础统计 | 详细分析 |
| **报告导出** | ❌ | ✅ PDF/Excel |
| **每日筛选** | ❌ | ✅ 改造后合规版 |
| **因子分析** | 基础 | 详细IC/IR分析 |
| **数据下载** | ❌ | ✅ |

### 3.2 功能详情

#### 3.2.1 策略回测

**功能描述：**
- 用户选择因子组合
- 设置回测参数（时间范围、初始资金、止盈止损）
- 运行回测
- 查看回测结果

**回测报告包含：**
- 总收益率
- 年化收益
- 最大回撤
- 夏普比率
- 胜率
- 盈亏比
- 持仓明细
- 交易记录

#### 3.2.2 因子分析

**功能描述：**
- 查看因子列表
- 查看因子IC/IR
- 查看因子相关性
- 分组收益分析

**内置因子：**

| 类型 | 数量 | 示例 |
|-----|------|------|
| 量价因子 | 12 | 动量、波动率、换手率 |
| 基本面因子 | 12 | PE、PB、ROE |
| 反转因子 | 8 | 5日反转、20日反转 |
| 技术因子 | 10 | RSI、MACD、KDJ |
| 其他因子 | 10 | 情绪、资金流向 |

#### 3.2.3 每日筛选（付费功能）

**功能描述：**
- 每日运行策略
- 输出筛选结果
- **改造后合规版**

**输出格式（改造后）：**
```json
{
  "date": "20260309",
  "filter_results": [
    {
      "ts_code": "601985",
      "name": "中国核电",
      "score": 8.5,
      "rank": 1,
      "factors": [
        {"name": "动量", "value": "9.2", "signal": "强势"},
        {"name": "RSI", "value": "66", "signal": "中性偏强"}
      ]
    }
  ],
  "disclaimer": "本系统基于历史数据分析，不构成投资建议"
}
```

**改造要点：**
- ❌ 不输出：价格、数量、金额、买入建议
- ✅ 只输出：代码、名称、得分、排名、因子分析
- ✅ 必须加：免责声明

## 4. 定价策略

### 4.1 套餐设计

| 套餐 | 价格 | 功能 | 限制 |
|-----|------|------|------|
| **免费版** | 0 | 回测基础功能 | 3次/月，1年周期 |
| **专业版** | 99元/年 | 全部功能 | 无限制 |

### 4.2 定价逻辑

**为什么是99元/年？**

| 对比 | 价格 | 相当于 |
|-----|------|--------|
| 聚宽专业版 | 500元/月 | 6000元/年 |
| 米筐专业版 | 300元/月 | 3600元/年 |
| **StockQuant** | **99元/年** | **8.25元/月** |

**定价策略：**
- 低价渗透，快速获客
- 年付锁定用户
- 后续可涨价（老用户不变）

### 4.3 付费转化

**转化路径：**
```
免费用户 → 用完3次回测 → 提示升级
                   ↓
              看到专业版优势 → 付费转化
```

**转化话术：**
- "您的免费回测次数已用完"
- "升级专业版，无限回测 + 每日筛选"
- "限时优惠：99元/年（原价199元）"

### 6.4 套餐权限控制

**实现方式：**

```python
# 装饰器：检查套餐权限
from functools import wraps
from flask import g

def check_subscription(f):
    @wraps(f)
    def decorated_function(*args, **kwargs):
        user = g.current_user
        plan = user.plan or 'free'
        
        # 检查回测次数
        if plan == 'free':
            count = get_monthly_backtest_count(user.id)
            if count >= 3:
                return error(message="回测次数已用完，请升级专业版")
        
        return f(*args, **kwargs)
    return decorated_function

def check_feature_limit(feature):
    """检查功能权限"""
    def decorator(f):
        @wraps(f)
        def decorated_function(*args, **kwargs):
            user = g.current_user
            plan = user.plan or 'free'
            
            # 每日筛选只有付费用户可用
            if feature == 'observer' and plan == 'free':
                return error(message="此功能仅限专业版用户使用")
            
            return f(*args, **kwargs)
        return decorated_function
    return decorator
```
